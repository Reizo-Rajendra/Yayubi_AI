# 🍽️ Catering Yayubi AI Chatbot

![Made With](https://img.shields.io/badge/made%20with-JavaScript-yellow)

AI Chatbot untuk membantu pelanggan mendapatkan informasi tentang **Catering Yayubi Semarang** secara cepat, interaktif, dan user-friendly 🤖✨

---

## 🚀 Features

- 🤖 Rekomendasi menu otomatis berdasarkan jenis acara  
- 📍 Informasi area layanan (Semarang & sekitarnya)  
- ⏰ Jam operasional (08.00 – 20.00 WIB)  
- 🧑‍🍳 Keunggulan catering (fresh, higienis, dll)  
- 💬 Chat responsif seperti customer service  
- 🌐 Mudah diintegrasikan ke website
